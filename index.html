Portofolio

Project Desain Web "Pondokku.Id

Pondokku.id adalah platform pendaftaran pesantren online yang memudahkan calon santri mendaftar tanpa harus antre. Dengan fitur pencarian pondok, formulir digital, dan notifikasi otomatis, Pondokku.id membantu proses pendaftaran jadi lebih cepat, efisien, dan terorganisir.

Project Desain Web "Vihvas"

Vihvas (Virtual Human Vascular) adalah proyek desain web berbasis AR/VR yang bertujuan mempermudah siswa dalam mempelajari organ tubuh manusia secara interaktif. Dengan visualisasi 3D dan pengalaman imersif, Vihvas membantu siswa lebih cepat memahami materi dan meningkatkan minat belajar di bidang anatomi.

Project Desain Web "Gametopia"

Gametopia adalah platform web yang menyediakan layanan top up untuk berbagai game mobile dan PC. Dilengkapi fitur pencarian game, pilihan metode pembayaran lengkap, serta desain yang responsif dan user-friendly, Gametopia memudahkan pengguna melakukan transaksi top up dengan cepat dan aman.

Project Pemrograman "BookStore"

Bookstore adalah aplikasi berbasis Java dasar yang dirancang untuk mensimulasikan sistem penjualan buku secara digital. Aplikasi ini memiliki fitur login, registrasi, pencarian buku, pembelian, dan proses pembayaran sederhana. Meskipun dibangun dengan konsep dasar, Bookstore mencerminkan alur kerja toko buku digital secara fungsional.

Project Pemrograman "Web Profile"

Personal Web Profile adalah proyek pemrograman web yang menampilkan halaman Home berisi perkenalan diri dan daftar keahlian (skill). Website ini dirancang dengan desain yang responsif sehingga tetap nyaman diakses melalui berbagai perangkat, baik desktop maupun mobile.
